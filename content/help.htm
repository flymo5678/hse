Help using this CD-Rom

How do I view PDF files?

You can download Acrobat Reader from the Adobe website.

If you already have Adobe Acrobat Reader installed on your computer, your Internet browser should launch the Acrobat plug-in automatically.

How do I view Microsoft PowerPoint files?

If you don't already have Microsoft PowerPoint installed, you must download Microsoft PowerPoint Viewer 2007.

If you already have Microsoft PowerPoint / PowerPoint Viewer installed on your computer, your Internet browser should launch the PowerPoint plug-in automatically.

How do I view Microsoft Word files?

If you don't already have Microsoft Word installed, you must download Microsoft Word Viewer 2003.

If you already have Microsoft Word / Word Viewer installed on your computer, your Internet browser should launch the Word Viewer plug-in automatically.

How do I view Microsoft Excel files?

If you don't already have Microsoft Excel installed, you must download Microsoft Excel Viewer 2003.

If you already have Microsoft Excel / Excel Viewer installed on your computer, your Internet browser should launch the Excel Viewer plug-in automatically.
